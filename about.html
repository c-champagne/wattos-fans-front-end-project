The purpose of Clear Dark Sky is to give users an easy way to learn about stargazing conditions based on location. This is accomplished by using two separate APIs, one that pulls location data and one that pulls weather data and astronomy info. At this time, Clear Dark Sky pulls data about your current location and combines it with your current weather to determine if viewing conditions are either excellent, good, or bad.
  • IP Geolocation
    IP Geolocation is the API our team chose for acquiring the user's current location. We initially chose this API because it also came with an Astronomy piece which we planned to also use. Further on into production, we realized this did not have the exact data we needed. To solve this problem, we found another API that provided more data about the moon.
  • World Weather Online
    World Weather Online was the second weather API our team worked with. While our first weather API gave us great data, we discovered WWO while trying to locate an API that also gave moon phase data. We realized we could keep things simple by switching over to using WWO for both our weather and moon data.
