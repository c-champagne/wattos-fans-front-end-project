  • The Team
The purpose of Clear Dark Sky is to give users an easy way of learning about skygazing conditions based on location. This is accomplished by using two separate APIs, one that pulls location data and one that pulls weather data and astronomy info.
  • IP Geolocation
    IP Geolocation is the API our team chose for acquiring the user's current location. We initially chose this API because it also came with an Astronomy API that gave us some helpful information.
